MVHub Open Core

The Core IIS Libraries

"Core" is a set of DLL; that permit the connectivity to a Multivalue Database thru IIS Web Server. Core normalize the web request information and pass it to a "Data Connector." Once, the required action is processed by the "MV Handler," Core is exploded into the value out to a requested web response.
